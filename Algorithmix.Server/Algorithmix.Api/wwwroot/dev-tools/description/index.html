Правила вёрстки страницы:
  • Все созданные Вами файлы должны быть заархивированы в файл с разрешением .zip (используйте данный шаблон в качестве примера)
  • Архив должен содержать строго определённую точку входа: HTML-файл с названием "X.html", где X - ID редактируемого алгоритма
  • Строки, выделенные зелёным, не рекомендуется удалять. Это необходимо для корректного отображения страницы на сайте
  • Рекомендуется использование стандартных стилей (шрифты, отступы и т. д.) для единого отображения всех страниц на сайте (css/styles.css)
  • Разрешается использование любых сторонних CSS- и JavaScript-библиотек
  • Удачи! :)
