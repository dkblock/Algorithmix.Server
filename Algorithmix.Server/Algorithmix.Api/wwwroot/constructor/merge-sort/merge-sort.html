Краткое описание сортировки
Исходная последовательность представляется как последовательный набор отдельных элементов (представлены разными цветами). Из подпоследовательностей формируем упорядоченные пары методом слияния, затем - упорядоченные четверки и так далее увеличивая шаг в 2 раза (каждая новая подпоследовательность после слияния выделяется одним цветом).
