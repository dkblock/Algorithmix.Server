Краткое описание сортировки
Массив разбивается на «отсортированный» подмассив (
- зеленый) и «несортированный» подмассив (
- синий). В начале отсортированный подмассив содержит только первый элемент нашего исходного массива. Каждый элемент (
- красный) в несортированном массиве оценивается так, что мы можем вставить его на свое место в отсортированном подмассиве. Вставка выполняется путем перемещения каждого элемента который больше (
- желтый), чем новый элемент, на одну позицию вправо. Это продолжается, пока весь наш массив не будет отсортирован.
