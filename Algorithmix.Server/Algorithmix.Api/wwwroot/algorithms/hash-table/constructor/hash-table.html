Хеш-таблица

Краткое описание
Случайно генерируются слова k, от которых будем считать хеш-функцию и вносить в хеш-таблицу. Входную строку посимвольно переводим в коды (Unicode) и конкатенируем. Далее считается хеш-функция (применяется метод решения коллизий двойным хешированием: H(k) = (h1(k) + i*h2(k)) mod m , где первый хеш берется по модулю длины таблицы, а второй по модулю длины таблицы - 1).  , i увеличивается на 1 и происходит вычисление очередной хеш-функции.
