Определение
Сортировка выбором — так же один из самых простых в реализации алгоритм сортировки. Как и в большинстве других подобных алгоритмов, в его основе лежит операция сравнения. Сравнивая каждый элемент с каждым, и в случае необходимости производя обмен, метод приводит последовательность к необходимому упорядоченному виду.
Сортировка массивов методом выбора, пожалуй, одна из самых простых для понимания, хотя и одна из самых медленных.
Алгоритм
  1. Начиная с элемента под индексом 0, ищем в массиве наименьшее значение.
  2. Найденное значение меняем местами с нулевым элементом.
  3. Повторяем шаги №1 и №2 уже для следующего индекса в массиве (отсортированный элемент больше не трогаем). 
Реализация
  1. function selectionSort(inputArr) {
  2.     let n = inputArr.length;
  3.     for (let i = 0; i < n; i++) {
  4.         let min = i;
  5.         for (let j = i; j < n; j++) {
  6.             if (inputArr[j] < inputArr[min]) {
  7.                 min=j;
  8.             }
  9.          }
  10.          if (min != i) {
  11.              let tmp = inputArr[i];
  12.              inputArr[i] = inputArr[min];
  13.              inputArr[min] = tmp;      
  14.         }
  15.     }
  16.     return inputArr;
  17. }
