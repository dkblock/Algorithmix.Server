Определение
Сортировка вставками — алгоритм сортировки, в котором элементы входной последовательности просматриваются по одному, и каждый новый поступивший элемент размещается в подходящее место среди ранее упорядоченных элементов.
Алгоритм часто используется при изучение компьютерных наук и обычно, является одним из первых алгоритмов сортировки, с которым знакомят учеников. Он интуитивно понятен и прост в реализации, но он очень медленный для больших массивов и поэтому редко используется для сортировки реальных данных, в основном только для небольших массивов информации
Сортировка вставками является устойчивым алгоритмом, который действительно хорошо работает только для почти отсортированных или небольших массивов. Сортировка вставками может использоваться на практике из-за ее эффективности для небольших (~ 10 элементов) наборов данных.
Алгоритм
  1. Массив разбивается на «отсортированный» подмассив и «несортированный» подмассив. В начале отсортированный подмассив содержит только первый элемент нашего исходного массива.
  2. Первый элемент в несортированном массиве оценивается так, что мы можем вставить его на свое место в отсортированном подмассиве.
  3. Вставка выполняется путем перемещения всех элементов которые больше, чем новый элемент, на одну позицию вправо.
  4. Пункты 1, 2, 3 повторяются, пока весь наш массив не будет отсортирован. 
Простой пример сортировки вставками
Реализация
  1. function insertionSort(arr) {
  2.     for (let j = 1; j < array.length; j++) {
  3.         let current = array[j];
  4.         let i = j-1;
  5.         while ((i > -1) && (array[i] > current)) {
  6.             array[i+1] = array[i];
  7.             i--;
  8.         }
  9.         array[i+1] = current;
  10.     }
  11. }
